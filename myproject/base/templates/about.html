📘 About This TODO Application

This TODO Management System is a simple and efficient task tracking web application developed using the Django framework. It allows users to manage their daily activities by creating, updating, completing, restoring, and deleting tasks easily. The project demonstrates how CRUD operations work in Django along with template rendering, database interaction, and task lifecycle management.

✨ Key Features

  • Create new tasks with title and description
  • Edit existing tasks anytime
  • Mark tasks as completed
  • Restore completed tasks back to active list
  • Move tasks to trash when no longer needed
  • Permanently delete tasks from trash
  • Manage multiple tasks efficiently from dashboard
  • Clean and user-friendly interface for better productivity

🛠 Technologies Used

  • Python – Core programming language
  • Django – Backend web framework
  • HTML – Page structure
  • CSS – Styling and layout design
  • SQLite – Default Django database

🎯 Project Objectives

The main objective of this project is to understand how Django handles database models, views, templates, and URL routing while building a real-world task management system. This application also demonstrates:

  • Django Models for database structure
  • Views for business logic
  • Templates for frontend rendering
  • URL routing for navigation
  • Task status handling (Active, Completed, Trash)

🚀 Learning Outcomes

By developing this project, important Django development concepts were practiced including:

  • CRUD operations implementation
  • Template inheritance usage
  • Navigation structure design
  • Database record management
  • Reusable UI components creation
  • Project structure organization

📌 Future Improvements

  • Add user authentication system
  • Implement task deadlines and reminders
  • Add search functionality
  • Introduce task priority levels
  • Deploy application online
